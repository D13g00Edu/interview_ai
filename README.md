# 🚀 InterviewCoach IA

AI-powered technical interview coach designed to help candidates improve how they communicate under pressure.

Many candidates lose great opportunities not because they lack technical knowledge, but because they struggle to explain ideas clearly, structure answers, and project confidence during interviews.

**InterviewCoach IA** was built to solve that.

---

## 🎯 What It Does

- Simulates realistic technical interviews with AI  
- Adapts questions based on seniority level  
- Evaluates depth and quality of responses  
- Provides structured feedback on communication  
- Detects recurring weaknesses over time  
- Helps convert weak patterns into strengths  

---

## 🧠 Why This Project Exists

Being technically strong is not always enough.

In many hiring processes, communication skills, clarity of thought, and confidence heavily influence outcomes.

This platform helps users practice in a safe environment and improve before the real interview.

---

## 🏗️ Architecture

The application follows a clean 4-layer architecture:

### 1. Frontend
- Next.js App Router  
- React Server Components  
- TypeScript  

### 2. Backend
- Next.js API Routes  
- Server-side business logic  
- Protected credentials handling  

### 3. External Services
- **Gemini AI** → Interview generation, evaluation, scoring, feedback  
- **Supabase** → PostgreSQL database, authentication, user history  

### 4. Infrastructure
- Vercel deployment  
- Edge CDN  
- HTTPS  
- Environment variables encryption  

---

## 🔐 Security

Security was a priority from day one.

- API keys never exposed to the browser  
- Gemini credentials live only on server-side routes  
- Frontend receives processed responses only  
- Supabase Auth for secure user sessions  
- Environment variables managed through Vercel
